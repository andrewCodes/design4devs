Images

"Images on your website serve much more of a purpose than just making your design look good."

1

You can put text on your images

Text can go directly onto an image provided the contrast between the image and the text is significant enough.

If the contrast is too low the text will be hard to read and your site will fail accessibility tests.

You can check for colour accessibility using this background image a11y check tool.

2

Overlay the image

A semi-transparent black overlay is the standard way of doing this. It darkens the image and allows light text to stand out.

You can also a colour to overlay an image. You should use the base colour of your website to ensure it fits with the overall design.

Another way to use an overlay is as a coloured gradient. Consider using a gradient from the dark version of your base colour to the light version of it.

3

Put text in a box

If you do not want to overlay your image to get text to stand out, you can use text boxes which sit on the image.

The box should have a slightly transparent background colour so the image can be made out through the box.

Doing so will make the image and the text boxes feel more like a single element.

4

Blur the image

This has a similar effect to an overlay and makes the text easier to read.

You can either blur the whole image or put text over a blurred section of an in-focus image.

If placing text over a blurred section of an image, be careful to ensure your text stays on that section on all screen sizes.

5

The floor fade

This is where we create a gradient overlay that runs from the top to the bottom of an image. The gradient runs from completely transparent to semi-transparent black.

As the image fades towards black at the bottom, light colour text is placed on the darker section.

When done well the floor fade can be almost unnoticeable.

Overlay the Image

.darken {
    background-image: linear-gradient(rgba(0,0,0,.5), rgba(0,0,0,.5)), url(YOUR IMAGE HERE);
}

This will put a semi-transparent black overlay on top of your background image so light text will be clearly readable even on a light image.

View example on CodePen

Put Text in a Box

.text-box {
    background-color: rgba(0,0,0,.25);
    color: #fff;
    display: inline;
    padding: 10px;
}

This will create a semi-transparent black text box with white text that you can place on a light image.

View example on CodePen

Blur the Image

.image-blur {
    background-image: url(YOUR IMAGE HERE);
    filter: blur(8px);
}

This will blur the background image. Try adjusting the size of the blur to see what looks best with your image.

View example on CodePen

The Floor Fade

.floor-fade {
    background-image: linear-gradient(to bottom, rgba(0,0,0,0), rgba(0,0,0,.6)), url(YOUR IMAGE HERE);
}

This will darken the background image towards the bottom. Light text can then be placed on the darkened section.

View example on CodePen
